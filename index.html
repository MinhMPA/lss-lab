lss-lab

Interactive visualization and animation tools for large-scale structure — cosmological simulations and galaxy surveys. Making the physics of the cosmic web tangible.

Tools

The Cosmic Web Sandbox→

An interactive, in-browser cosmic web you reshape with cosmological dials. Gaussian field, FFTs, and Lagrangian perturbation theory run client-side, so the controls respond in real time.

Galaxy Size — LSS Correlation Explorer→

Member galaxy sizes inside dark-matter host halos, correlated with the large-scale gravitational potential well and density perturbations — the size analog of intrinsic alignments. Couplings respond live.
